Course Overview

This course provides a complete introduction to Cloud Computing and DevOps using AWS. Over 5 days, students will learn cloud fundamentals, AWS services, networking, DevOps practices, CI/CD pipelines, monitoring, and deployment of applications.

Syllabus

Day 1: Cloud Foundations & AWS Introduction

  • Cloud fundamentals, service models (IaaS, PaaS, SaaS)
  • Cloud providers & AWS introduction
  • AWS IAM, Regions & Availability Zones

Day 2: Core AWS Services

  • Amazon EC2 (compute)
  • Amazon S3 (storage)
  • Database basics (SQL vs NoSQL, RDS, DynamoDB)

Day 3: Networking & DevOps

  • Networking basics (IP, subnets)
  • Amazon VPC & security
  • DevOps lifecycle and principles

Day 4: Version Control & CI/CD

  • Git & GitHub basics
  • Containerization & orchestration
  • CI/CD pipelines (GitHub Actions / Jenkins)

Day 5: Monitoring & Project

  • Amazon CloudWatch (monitoring)
  • AWS Lambda, SNS, CloudFront
  • Mini project deployment

Schedule

Duration: 5 Days
